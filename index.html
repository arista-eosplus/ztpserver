Arista EOS+ ZTPserver

This is the official Arista EOS+ ZTPServer GitHub page.

ZTPServer provides a robust server side solution targeted at providing a comprehensive bootstrap solution for Arista EOS network elements.

ZTPServer Highlights

  • Lightweight HTTP server implementation for EOS ZTP provisioning
  • Extends basic capability of EOS ZTP to allow more robust provisioning activities
  • Extensible for easy integration into network operational environment
  • Can be run natively in EOS
  • Arista-led community open source project

ZTPServer Features

  • Automated configuration generation
  • Automated system software deployment
  • Topology based provisioning and validation
  • ZTR (Zero Touch Replacement)
  • XMPP integration for logging and control

System requirements

  • python 2.7
  • dhcpd
  • pip
  • git

To grab the latest release:

$ git clone https://github.com/arista-eosplus/ztpserver.git -b release-1.0
$ python setup.py build
$ python setup.py install
